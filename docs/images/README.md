## How to Create Screenshots

1. **Open Obsidian** with the template installed
2. **Create a sample dataset** using the template
3. **Take screenshots** during the process:
   - macOS: Cmd+Shift+4
   - Windows: Snipping Tool or Win+Shift+S
   - Linux: Screenshot tool or Shift+PrtScr
4. **Annotate if needed** (optional):
   - Highlight important areas
   - Add arrows or labels
   - Use tools like GIMP, Photoshop, or online editors
5. **Optimize file size**:
   - Use PNG format
   - Compress to <200KB per image
   - Tools: TinyPNG, ImageOptim, or GIMP
6. **Save with descriptive names**:
   - `template-prompts.png`
   - `styled-output.png`
   - `before-after.png`

## Image Guidelines

- **Format**: PNG (preferred for screenshots)
- **Size**: Balance quality and file size (<200KB each)
- **Content**: Clear, readable text at 100% zoom
- **Privacy**: No personal information visible
- **Theme**: Show both light theme (primary) and dark theme (if needed)
- **Language**: English UI/text
- **Annotations**: Use SDCStudio colors if adding overlays:
  - Primary: #0a2342
  - Secondary: #2ca58d
  - Accent: #f0a500
